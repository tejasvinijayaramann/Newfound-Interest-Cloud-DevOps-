## Why DevOps?

My interest in DevOps started when I began exploring computer networking through the CCNA learning path. While learning networking concepts and understanding how systems communicate, I discovered the world of Cloud Computing and DevOps.

What started as curiosity about networking gradually evolved into a deeper interest in how applications are built, deployed, automated, monitored, and maintained at scale. The combination of networking, cloud technologies, automation, and problem-solving is what attracted me to DevOps.

This repository documents my journey as I learn and build skills in Linux, Networking, AWS, Docker, Kubernetes, CI/CD, and other DevOps technologies. My goal is to gain hands-on experience, build practical projects, and grow into a Cloud/DevOps Engineer.
